# RealsensePack
Intel Realsense Support Pack

当前支持的设备有：
Intel Realsense D430/SR300

预设的深度图为 16 位，640x480，30fps
预设的灰度图为 8 位，640x480，30fps

支持的系统及环境：
- Windows 10 及以上
- Ubuntu 20.04 LTS 及以上（其他的 Linux 发行版系统我没测试过）
- Python `3.10`, `3.11`

执行下面的指令安装
```
python setup.py build
python setup.py install
```
